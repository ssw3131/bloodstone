dk.isDebug / log

			
개발 작업 시 debug 모드를 활성화 해서 개발합니다. isDebug 가 true 일 경우에만 log 함수가 작동 됩니다.

			
log 함수는 console.log를 대체합니다.
log 함수에 첫번째 파라미터가 'debug' 일 경우 debugger; 중단점이 실행됩니다.

			
				dk.isDebug( true );
				log( 'hi' );
				log( 'debug', 'hi' );

UTIL

			
유틸리티 함수들


			

				
random

				
특정범위 내에 랜덤숫자를 반환합니다.
max 기본값 : 1, min 기본값 : 0

				
					// min ~ max
					dk.random( max, min );

					// 0 ~ max
					dk.random( max );

					// 0 ~ 1
					dk.random();
				

			

			
			

				
randomInt

				
특정범위 내에 랜덤정수를 반환합니다.
min 기본값 : 0

				
					// min ~ max
					dk.randomInt( max, min );

					// 0 ~ max
					dk.randomInt( max );
				

			

			
			

				
randomColor

				
랜덤 컬러값을 반환합니다.

				
					dk.randomColor();
				

			

			
			

				
timeCheck

				
timeCheck 함수 실행 사이의 시간을 반환합니다.

				
					dk.timeCheck();
					// ...
					dk.timeCheck();
				

			

			
			

				
parseLocation

				
location.search의 문자열을 & 문자열로 나눠 오브젝트에 key, value 값으로 저장해 반환합니다.

				
					// index.html?a=1&b=2;
					dk.parseLocation();
					// {a: "1", b: "2"}
				

			

			
		

		

		

			
dk.ListManager

			
리스트의 length 값을 받아 리스트 안의 활성화 id 값을 관리합니다.

			
활성화 id 값을 바꾸고 현재 id와 이전 id 값을 제공합니다. autoPlay 기능을 제공합니다.

			
배열의 id, 슬라이드의 id, 버튼리스트의 id 등에 활용 가능합니다.

			
				/**
				 * @class	: ListManager
				 * @param	: leng - list length
				 * @param	: act - 활성화 함수
				 * @param	: option - initId, infinity, autoPlay, autoPlaySpeed
				 */
				dk.ListManager( leng, act, option );
      

			
				dk.ListManager( 4, function( id, oldId, directon ){}, {
					initId: 0,
					infinity: true,
					autoPlay: false,
					autoPlaySpeed: 4
				} );
      

			

				
leng

				
Type : int

				
leng 이 없거나 0 이면 안됩니다.

			

			
			

				
act

				
Type : function

				
act 는 필수항목 입니다.

				
활성화 id 값이 바뀔때 실행될 리스너 함수입니다.

			

			
			

				
option

				

					
initId

					
Type : int

					
초기 설정 될 id 값

				

				

					
infinity

					
Type : boolean

					
Default : true

					
autoPlay 시 무한반복 설정

				

				

					
autoPlay

					
Type : boolean

					
Default : false

					
autoPlay 설정

				

				

					
autoPlaySpeed

					
Type : number

					
Default : 4

					
autoPlay 시간 설정 (초)
